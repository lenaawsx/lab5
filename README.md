 Лабораторная работа №5
 ### Какие интерактивности добавлены на страницу?
- Кнопка «Наверх», которая появляется при прокрутке вниз и плавно возвращает на начало страницы.  
- Аккордеон с описанием проектов, открывающий и закрывающий текст по клику.  
- Фотогалерея с фильтрацией по категориям (Природа, Город, Все).  
- Модальное окно для увеличения изображений галереи.  

### Где использовался ИИ, а где код писался вручную?
- Код кнопки «Наверх», аккордеона, галереи с фильтрацией и модалки был сгенерирован с помощью ИИ (ChatGPT).  
- Вёрстка страницы, тексты описаний проектов и стили (CSS) писались вручную.  

### Что показалось самым интересным или сложным?
- Интересно было реализовать фильтрацию фотогалереи так, чтобы при выборе категории показывались только соответствующие картинки.  
- Сложнее всего было совместить аккордеон с галереей и модальным окном, чтобы всё работало без конфликтов.  

### Какие запросы делали LLM?
- «Напиши JavaScript, который добавляет кнопку 'Наверх'. При прокрутке страницы вниз она появляется, при нажатии — плавно прокручивает вверх.»  
- «Сделай аккордеон на чистом JS: несколько заголовков, при клике открывается и закрывается блок текста.»  
- «Создай фотогалерею с фильтрацией по категориям. При выборе категории показывай только соответствующие фото.»  
- «Добавь обработчик: при клике на фото открывается модальное окно с увеличенным изображением и затемнением фона, при клике по крестику или фону — закрыть.»
